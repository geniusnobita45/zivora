# Security Policy

## Supported Version

Security fixes are supported for the production ZIVORA AI website deployed from the main branch.

## Reporting a Vulnerability

Please report suspected vulnerabilities privately to the repository owner or site operator. Include the affected URL, reproduction steps, impact, and any relevant request or response details.

Do not publicly disclose active vulnerabilities before the owner has had a reasonable opportunity to validate and remediate them.

## Handling

The owner should acknowledge reports, assess severity, remediate confirmed issues, rotate any exposed credentials when needed, and deploy fixes through the normal production release process.
